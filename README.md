## ğŸ§  How it Works (RAG Pipeline)

1. **Parse PDFs** using `pdfplumber` / `PyPDF2`
2. **Chunk text** into overlapping segments
3. **Generate embeddings** using `all-MiniLM-L6-v2`
4. **Store & search** embeddings using **FAISS L2 index**
5. Retrieve top-k relevant chunks
6. Build a clean context-prompt
7. Generate answer using **Qwen2.5-0.5B-Instruct**
8. Display final result + page references
